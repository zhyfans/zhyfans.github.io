mysql驱动与mysql版本及jdk版本对应

2019-05-30 #开发 #数据库 #mysql

今天新启一个项目的时候报错，日志显示数据库驱动问题，发现是使用驱动版本与数据库版本及jdk版本不一致造成的，记录一下。

修改之前，本地环境为：

jdk：1.7
mysql版本：8.0
mysql驱动版本：mysql-connector-java-8.0.16.jar

启动项目报错，看了官网文档之后，把驱动版本修改为：mysql-connector-java-5.1.47.jar 解决问题

下面是版本对应：

官网文档地址：

https://dev.mysql.com/doc/connector-j/5.1/en/connector-j-versions.html （Connector/J Versions 5.1）

https://dev.mysql.com/doc/connector-j/6.0/en/connector-j-versions.html （Connector/J Versions 6.0）

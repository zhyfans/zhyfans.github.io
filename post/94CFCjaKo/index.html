Linux配置NFS共享

2019-05-27 #linux

NFS（Network File System）网络文件系统
服务可以将远程 Linux 系统上的文件共享资源挂载到本地主机的目录上，从而使得本地主机(Linux 客户端)基于 TCP/IP 协议，像使用本地主机上的资源那样读写远程 Linux 系统上的共享文件。

一、服务器配置
通过SSH工具连接服务器

输入命令：vi /etc/exports 按i编辑
加入配置内容：共享目录 客户机IP(权限ro-只读 rw-读写)

例如：/picclife/peflexnfsdta/cbsnfsdta 10.135.100.146(ro)

按esc退出编辑，输入:wq保存配置
输入命令：
service rpcbind restart
service nfs restart
exportfs
重启NFS服务，使配置生效
查看挂载情况：输入命令：cat /etc/exports

二、客户机挂载

通过SSH连接客户机挂载服务器共享目录
输入命令：mount 服务器IP:服务器共享目录 本地目录

例如：mount 10.135.100.26:/picclife/peflexnfsdta/ /picclife/peflexnfsdta

进入本地相应目录，可以看到服务端共享目录中的文件。
